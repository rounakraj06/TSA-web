TSA Web

TSA Web is an educational platform developed for students of Classes 8 to 10.

Features

- Study Notes
- Previous Year Question Papers (PYQs)
- Subject-wise Learning Resources
- Easy Navigation
- Mobile Friendly Design

Purpose

The platform helps students prepare effectively for examinations by providing notes and previous year question papers in one place.

Technologies Used

- HTML
- CSS
- JavaScript
- PHP
